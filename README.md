# Gradient Text Effect using only HTML & CSS | CodingLab

Um projeto simples de **efeito de texto com gradiente** usando apenas HTML e CSS.

## ✨ Sobre

Este projeto demonstra como criar um **texto moderno com efeito gradient**, ideal para títulos e destaques em sites.

## 🛠️ Tecnologias

- HTML
- CSS (gradient, background-clip)

## 🎯 Funcionalidades

- Texto com efeito gradiente
- Design moderno
- Fácil de aplicar em qualquer projeto

## 🚀 Como usar

1. Baixe ou clone o repositório  
2. Abra o arquivo `index.html` no navegador
